## TO RUN APP:
1. `pip install -r requirements.txt`
2. `python app.py`

## TO SETUP REPO:
- `git init`
- `git remote add origin https://github.com/username/repository.git`
- `git remote -v` - check to see if it's alright
- `git add .`
- `git commit -m "Initial commit"`
- `git push -u origin master`
